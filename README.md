# StyleTTS2 UK ONNX Inference

Цей проект забезпечує оптимізований інференс моделі **StyleTTS2 (Ukrainian Multi-speaker)** через **ONNX Runtime**. Це рішення дозволяє генерувати якісний український голос з низькою затримкою на CPU, що ідеально підходить для ноутбуків з обмеженими ресурсами.

## 🚀 Основні переваги

* **Швидкість:** Використання ONNX Runtime забезпечує швидкий інференс без навантаження повноцінним PyTorch.
* **Ефективність:** Оптимізовано для CPU (використання багатопоточності).
* **Стерильність:** Скрипт налаштований на повне придушення системних попереджень для чистого виводу в консолі.

## 🛠 Технічні характеристики

* **Модель:** StyleTTS2 (українська мультиспікерна версія).
* **Формат:** `.onnx` (opset 19).
* **Декодер:** Вбудований HiFi-GAN.

## 📂 Структура проекту

```text
styletts2-uk-onnx-inference/
├── models/
│   └── styletts2.onnx      # Файл моделі (помісти сюди)
├── voices/
│   └── Інна Гелевера.pt    # Файл пресету голосу
├── main.py                 # Основний скрипт для генерації
└── requirements.txt        # Список залежностей

```

## ⚙️ Як налаштувати

### 1. Підготовка середовища

Клонуй репозиторій та створи віртуальне середовище:

```bash
python -m venv .venv
# Windows:
.venv\Scripts\activate
# Linux/macOS:
source .venv/bin/activate

```

### 2. Встановлення бібліотек

```bash
pip install -r requirements.txt

```

### 3. Додавання файлів

* Скопіюй свій `styletts2.onnx` у папку `models/`.
* Скопіюй `Інна Гелевера.pt` у папку `voices/`.

### 4. Запуск

```bash
python main.py

```

## 💡 Поради щодо оптимізації

* **Чистота консолі:** Якщо ви хочете повністю приховати попередження сторонніх бібліотек на системному рівні, можна запускати скрипт так:
`python -W ignore main.py`
* **Продуктивність:** Скрипт автоматично використовує CPU. Якщо на ноутбуці виникає перегрів, переконайтеся, що система охолодження чиста, а параметри ONNX Runtime налаштовані на оптимальну кількість потоків (за замовчуванням у бібліотеці).
